Solana × AI Agents

Imperial AI Agent Hackathon — Solana x CoralOS Track
An open-source MCP (Model Context Protocol) server that enables AI agents to autonomously interact with the Solana blockchain — wallets, SPL tokens, DeFi, and on-chain data.
GitHub Repo →

The Problem

AI agents can reason. They can browse the web. They can write code.
But they cannot meaningfully interact with blockchains out of the box.
  • No standard protocol for agent ↔ blockchain interaction
  • Each agent framework needs bespoke Solana integration
  • On-chain operations require careful safety handling
  • DeFi data (prices, quotes) is fragmented across APIs
Result: agents that could earn on Solana are locked out by integration complexity.

SOL Forge: The Plumbing

A standardized MCP server that any agent can plug into and immediately start transacting on Solana.
6

Wallet Ops

Create wallets, check balances, transfer SOL, airdrop. Agents manage their own funds.

6

Token Ops

Create SPL tokens, mint, transfer, burn, query. Full token lifecycle.

5

Explorer

Read transactions, accounts, blocks, signatures. Understand on-chain state.

4

DeFi

Real-time prices via Jupiter, swap quotes, multi-token pricing.

20 total tools · TypeScript · MIT license
